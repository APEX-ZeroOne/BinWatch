# BinWatch

일경험 프로젝트형 사업 프로젝트 - AI 스마트 분리수거 쓰레기통 관련한 앱 개발입니다.

해당 앱은 다음과 같은 기능을 가지고 있습니다.

1. 카메라로 현재 쓰레기통 내부 상황 확인 가능
2. 분류 현황 확인
3. 쓰레기통 잔여 용량 확인
4. 에러 로그 확인 - 분류 실패한 쓰레기가 있는지
